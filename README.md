## Deployment instructions [WIP]

### Docker Container

The server can be built as a Docker image using the script `scripts/build-docker.bat` (Windows only) or executing the following command on the project's root folder:

> `docker build -t xxxx .`

Once built, it can be accessed through Docker Desktop or by using the following command:

> `docker run -p 8000:8000 xxxx`
